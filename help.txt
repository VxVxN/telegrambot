📋 Бот для управления задачами

Основные команды:
• help / хелп - показать эту справку
• list / список - показать задачи на сегодня
• full list / полный список - показать все задачи
• add [дата?] [текст] / добавить [дата?] [текст] - добавить новую задачу
• delete [ID] / удалить [ID] - удалить задачу
• clear / очистить - удалить все задачи

Команды для повторяющихся задач:
• /repeat_daily [ID] - повторять задачу ежедневно
• /repeat_weekly [ID] [дни] - повторять по дням недели
• /repeat_custom [ID] [интервал] - повторять каждые N дней

Команды для криптовалют:
• btc / /btc - показать текущую цену Bitcoin (BTC)
• eth / /eth - показать текущую цену Ethereum (ETH)
• xrp / /xrp - показать текущую цену XRP
• цены / /prices - показать цены всех криптовалют
• подписаться / /subscribe - подписаться на ежедневные уведомления в 10:00
• отписаться / /unsubscribe - отписаться от уведомлений

Примеры задач:
• добавить Купить молоко (дата будет сегодняшней)
• добавить 25.12.2023 Купить подарки
• /repeat_weekly 1 monday wednesday friday
• /repeat_custom 1 5

Примеры криптокоманд:
• btc - получить текущую цену Bitcoin
• цены - получить цены всех монет
• подписаться - включить ежедневные уведомления в 10 утра

📌 Важно:
• ID задачи показывается при её создании и в списке задач
• Для подписки на уведомления о ценах используйте команду "подписаться"
• Уведомления о ценах приходят каждый день в 10:00 по вашему времени
• Цены обновляются в реальном времени через CoinGecko API
